# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main | ✅ |

## Reporting a Vulnerability

If you discover a security issue in this repository, please open a GitHub issue or contact the repository owner.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Security Best Practices

- Never commit API keys.
- Store secrets in `.env` files.
- Validate external inputs.
- Review permissions before using AI agents.
- Keep dependencies up to date.
